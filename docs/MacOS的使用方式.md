# MacOS 使用方式

1. 打开 系统偏好设置
2. 选择 网络
3. 左侧选择你上网正在使用的设备，然后在右侧点击 高级
4. 选择 代理
5. 左边选择 SOCKS 代理，右边输入 `127.0.0.1`和你在配置文件中设置的端口号。如果是其他机器想要访问一个远端的agent，那么`127.0.0.1`需要改成远端agent的地址
6. 点击“好”（网络那里也要点“应用”）
